Installation

Get started.

Claude Code: Add the Marketplace + Install Plugins

In Claude Code, run:

# Add the marketplace /plugin marketplace add danielraffel/generous-corp-marketplace # Install plugins (pick the ones you want) /plugin install worktree-manager@generous-corp-marketplace /plugin install chainer@generous-corp-marketplace /plugin install prompt-repeater@generous-corp-marketplace /plugin install orchestrate@generous-corp-marketplace /plugin install design-partner@generous-corp-marketplace /plugin install juce-dev@generous-corp-marketplace # Restart Claude Code to load plugins

OpenAI Codex: Install Skills

Codex scans for skills in repo or user locations under .agents/skills. Install user-wide via symlink:

git clone https://github.com/danielraffel/generous-corp-marketplace cd generous-corp-marketplace mkdir -p ~/.agents/skills ln -s "$(pwd)/skills/claude" ~/.agents/skills/claude

Claude Code: Install Skills

Claude Code loads skills from ~/.claude/skills/ or .claude/skills/. Install via symlink:

git clone https://github.com/danielraffel/generous-corp-marketplace cd generous-corp-marketplace mkdir -p ~/.claude/skills ln -s "$(pwd)/skills/claude" ~/.claude/skills/claude
